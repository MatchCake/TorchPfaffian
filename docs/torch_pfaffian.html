Module contents¶

TorchPfaffian is a Python package for efficiently computing the Pfaffian of skew-symmetric matrices using PyTorch.

torch_pfaffian.get_pfaffian_function(name: str = 'PfaffianFDBPf') → Callable[[Tensor], Tensor]¶
torch_pfaffian.pfaffian(matrix: Tensor, *, sign: bool = True, check_input: bool = False) → Tensor¶

Compute the Pfaffian of a skew-symmetric matrix, choosing the strategy from the input.

The matrix has shape (..., 2n, 2n) and the result has shape (...,), sharing the backend, dtype, and device of matrix.

Strategy selection:

The Rust kernel runs on CPU, so a non-CPU (e.g. CUDA) input is routed to PfaffianParlettReid, which runs natively on the input device and avoids a host round-trip.

The Pfaffian is only defined for skew-symmetric matrices; the strategies assume this and do not check it. Pass check_input=True to validate the assumption. For large matrices the Pfaffian can exceed the floating range and overflow to inf; a RuntimeWarning is emitted when the result is not finite.

Parameters:

  • matrix – Skew-symmetric matrix of shape (..., 2n, 2n).

  • sign – When True (default) return the signed Pfaffian, otherwise its magnitude.

  • check_input – When True, validate that matrix is square in its last two dimensions and skew-symmetric (A == -A^T) before computing, raising ValueError otherwise. Off by default (False) so trusted inputs pay nothing; the check is an O(n^2) comparison, cheap relative to the O(n^3) Pfaffian.

Returns:

The Pfaffian of the input, of shape (...,).

Return type:

torch.Tensor
